0. Introduccin

El Modelo de Regresin Logstica es un algoritmo de clasificacin que lo utilizamos para poder predecir la probabilidad de una variable dependiente categrica, en este caso la variable que queremos predecir es si se comete fraude o no.Este algoritmo se basa principalmente en el clculo de logaritmos para hallar el error.

6.3. Curva ROC¶

Para evaluar nuestro modelo, la representaremos graficamente para deducir que tan bueno es el modelo.

In [14]:
# keep probabilities for the positive outcome only
yhat = predict_probabilities[:, 1]
# calculate roc curves
fpr, tpr, thresholds = roc_curve(y_val, yhat)
# plot the roc curve for the model
plt.plot([0,1], [0,1], linestyle='--', label='No Skill')
plt.plot(fpr, tpr, marker='.', label='Logistic Regression')
# axis labels
plt.xlabel('False Positive Rate')
plt.ylabel('True Positive Rate')
plt.legend()
# show the plot
plt.show()

# Display the ROC AUC.

print('The Compute Area Under the Receiver Operating Characteristic is '+ str(roc_auc_score(y_val,yhat)))

La curva ROC nos muestra un resultado muy bueno pero realmente atpico. Esto se debe a que los datos estn desbalanceados, por lo que no utilizaremos esta mtrica para basar la principal diferencia con los dems modelos.
